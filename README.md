# MyCoverAI Test

## Installation

1.  Clone the repo

    ```sh
    git clone https://github.com/Emmanuel0705/mycoverai.git
    ```

---

- Rename example.env to .env and update the content

2.  RUN

    ```sh
    docker-compose up -d --build
    ```

3.  RUN
    ```sh
    docker logs mycover_test -f
    ```
